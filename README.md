# HOI4 Gemini Bot

A Gemini bot to assist with Hearts of Iron IV gameplay and strategy.

## Overview

This project provides an AI-powered bot built with Google's Gemini API that offers strategic guidance, gameplay tips, and assistance for Hearts of Iron IV players.

## Features

- **Strategy Assistance**: Get recommendations for military and diplomatic strategies
- **Gameplay Tips**: Learn optimal playstyles for different nations
- **Resource Management**: Advice on industrial production and resource allocation
- **Technology Guidance**: Navigate the tech tree efficiently
- **War Planning**: Strategic military doctrine and formation guidance

## Getting Started

### Prerequisites

- Python 3.8 or higher
- Google Gemini API key
- Hearts of Iron IV (for context and reference)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/Storm-bit-star/hoi4-gemini-bot.git
cd hoi4-gemini-bot
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Set up your Gemini API key:
```bash
export GEMINI_API_KEY='your-api-key-here'
```
